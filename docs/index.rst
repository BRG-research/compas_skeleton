Introduction
============

`COMPAS Skeleton <https://github.com/BlockResearchGroup/compas_skeleton>`_ (compas_skeleton) is a mesh modeling package for the `COMPAS <https://compas-dev.github.io/>`_ framework.
It generates a coarse(low-poly) or a high resolution mesh from a set of branch lines while allowing interactive user input.
It offers efficient modification methods for the resulting mesh, by modifying the input branch lines or changing skeleton parameters (such as width etc).


Applications
============

Skeleton provides a way of sketching 2D diagrams and meshes from minimal inputs.
For example, it can be used to create force patterns for `RhinoVault <https://blockresearchgroup.github.io/compas-RV2>`_ and `TNA <https://blockresearchgroup.github.io/compas_tna>`_.
